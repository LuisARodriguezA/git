### Recursi√≥n
* No es muy usada en automatizaci√≥n pero si en general.
* In programming, recursion is a way of doing a repetitive task by having a function call itself.
	*  Es una funci√≥n que se llama a si misma hasta llegar a un base case que es el caso final, la ultima mu√±eca rusa

#### Ejemplo recursi√≥n
```run-python
def is_power_of(number, base):
  # Base case: when number is smaller than base.
  if number < base:
    # If number is equal to 1, it's a power (base**0).
    return number == 1

  # Recursive case: keep dividing number by base.
  return is_power_of(number/base, base)

print(is_power_of(8,2)) # Should be True
print(is_power_of(64,4)) # Should be True
print(is_power_of(70,10)) # Should be False
```
